Projects

OCR Chatbot

The idea of integrating NLP in chatbot to generate images from text prompts faces several challenges, including issues with resolution and complexity, as generating high-quality, detailed images is computationally expensive. There are also challenges with limited datasets, which can hinder effective training of the models, as well as the need for significant computational resources for image generation. Ensuring data efficiency to prevent over-fitting and improving generalization so the chatbot can generate diverse images for unseen prompts are additional hurdles. Overcoming these challenges is key to improving the chatbot's ability to generate accurate and creative images from text.

Travel Website

The interactive travel website, developed in collaboration with Elewayte Corporate using HTML, CSS, JavaScript, and PHP, allows users to explore destinations, view shared travel experiences, and arrange budget plans. Key features include a platform for travelers to share stories and ratings, tools for browsing destinations based on preferences, and a budget planner to estimate trip costs. The system also includes a ticket registration feature integrated with a backend database for managing bookings. This project combines dynamic content, user engagement, and practical tools, with potential for future enhancements such as AI recommendations, real-time integrations, and personalized user profiles.

Automation Attendance Tracking

The project combines deep learning and IoT to create an automated face recognition-based attendance system. IoT devices (like cameras) capture images of individuals in a room, and deep learning models detect and recognize faces. The system compares captured faces to a pre-existing database, marking attendance in real time. The backend processes images, updates attendance records in a database, and provides a user interface for viewing reports. Technologies like TensorFlow, OpenCV, and FaceNet are used for face detection and recognition, while IoT devices like Raspberry Pi handle image capture. This system improves time efficiency, accuracy, and scalability while reducing manual effort and errors in attendance marking.

Phishing Website Detection

Phishing website detection identifies fraudulent websites designed to steal sensitive user information by mimicking legitimate ones. The system uses techniques like URL analysis to detect suspicious patterns, content analysis to identify phishing elements in HTML or scripts, and machine learning models to classify websites based on features such as URL structure, SSL certificates, and domain age. Blacklisting, whitelisting, and behavioral analysis further enhance detection. Common tools include libraries like Scikit-learn, TensorFlow, and datasets like PhishTank. Despite challenges like evolving phishing techniques, false positives, and ensuring real-time accuracy, phishing detection is crucial for browser extensions, email filters, and enterprise security systems.
